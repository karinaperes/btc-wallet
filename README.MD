## Criação de uma carteira de criptomoedas

Exercício aplicado pelo curso da DIO sobre Blockchain

Clone o repositório

Instale as dependências

`**npm install**`


Rode o código, dentro da pasta raiz:

`**./src/createWallet.js**`

Será exibido no terminal:
```terminal
Endereço
Chave Privada
Seed
```
